Comandos para o terminal:

dir ou ls         => Lista arquivos da pasta
cls ou ctrl + l   => Limpa a tela
cd                => Entra no diretório 
cd ..             => Volta para a pasta anterior
cd~               => Volta ao início
mkdir             => Cria um pasta
rmdir             => Exclui a pasta
touch nome-arquivo=> Cria um arquivo, mas não entra nele
nano nome-arquivo => entra nesse arquivo (Para sair do nano e salvar - CTRL +X, Y, e enter)
mv                => Move ou renomeia arquivos/pastas
mv -d             => -d quer dizer diretório
ls                => mostra todas as pastas e arquivos do diretório
ls -l             => mostra os 3 arquivo/pastas do diretório em ordem e detalhes da ultima alteração.
ls -lh            => Mostra todas os arquivos e pastas do dirtório em ordem e detalhes.
ls -a             => Mostra pastas ocultas


Exemplos:

mkdir nome-da-pasta
rmdir nome-da-pasta
mv nome-original nome-que-deseja
